# Space Invaders (Raylib + C++)

A small **Space Invaders–style game** written in **C++** using the **Raylib** graphics library.
This project was created as a simple practice project to learn basic game development concepts such as rendering, input handling, game loops, and collision detection.

---

## Features

* Player-controlled spaceship
* Enemy waves moving across the screen
* Shooting mechanics
* Basic collision detection
* Score tracking
* Simple and clean game loop

---

## Technologies

* **C++**
* **Raylib**

---

## Requirements

Before building the project, make sure you have:

* **C++ compiler** (g++, clang, or MSVC)
* **Raylib installed**


---


## Controls

| Key                    | Action         |
| ---------------------- | -------------- |
| **Left / Right Arrow** | Move spaceship |
| **Space**              | Shoot          |
| **Esc**                | Exit game      |

---


## Learning Goals

This project demonstrates:

* Basic **game loop structure**
* **Player input handling**
* **2D rendering**
* **Simple collision detection**
* Managing **game objects**
